# BandLingo
A Dictionary Of Xiaomi Vela Band/Watch

BandLingo是一款专为小米vela穿戴设备（手环/手表快应用）打造构建和使用，利用碎片化时间，在腕上轻松记忆单词。

# 构建和使用
如果您是普通用户，想使用BandLingo，请去表盘自定义/AstroBox/米坛社区自行搜索下载RPK包安装。
但如果您是开发者，您可以克隆该仓库到您的设备上，打开 AIoT-IDE 运行 npm -i 即可开始构建的微型语言学习与词汇辅助工具。

# 词典的使用
为了方便大家共同完善 BandLingo 的内容生态，我们现将原本属于 Pro 版本的核心词库完全开源。
您可以自由地将这些词库用于二次开发、学术研究，或在手环端进行离线导入
词库数据部分同样随本项目以 GPLv3 形式开源。这意味着如果你导出了本词库用于其他查词/背单词软件，你的软件也必须保持开源。
